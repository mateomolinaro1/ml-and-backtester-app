## 1. Create a `.env` file

Before running the project, create a `.env` file at the **root of the repository**.


The `.env` file should contain the following environment variables:

```env
WRDS_USERNAME=
WRDS_PASSWORD=

IB_HOST=
IB_PORT=
IB_CLIENT_ID=

BUCKET_NAME=ml-and-backtester-app
AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
AWS_DEFAULT_REGION=
OUTPUT_FORMAT=
